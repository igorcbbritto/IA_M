# Marvel Zombies Companion

Aplicação web de referência rápida para **Marvel Zombies: Zombicide**, criada para auxiliar jogadores durante as partidas com regras, mecânicas, combate, sistema de fome, inimigos e expansões.

## Objetivo

O projeto foi desenvolvido para centralizar informações importantes do jogo em uma interface simples e rápida de consultar durante a partida.

## Funcionalidades

- Consulta de regras do jogo base
- Referência rápida de combate
- Sistema de fome
- Informações sobre inimigos
- Resumo de expansões
- Interface pensada para consulta prática

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript

## Como executar

1. Baixe ou clone este repositório
2. Abra o arquivo `index.html` no navegador
